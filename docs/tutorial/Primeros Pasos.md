# Comenzando con SimplestColabDocs

SimplestColabDocs es una herramienta de documentación colaborativa simple para sitios estáticos.

## Características Principales

- Sin librerías externas de JavaScript
- Estructura en árbol con categorías
- Soporte completo de Markdown
- Tema claro y oscuro
- Búsqueda de texto completo
- Colaboración mediante Git Pull Requests

## Cómo Agregar un Documento

1. Añade tu archivo `.md` en la carpeta `docs` (o una subcarpeta)
2. Si tu documento contiene imágenes, colócalas en una carpeta con el mismo nombre que tu documento (sin .md)
3. **Push a la rama principal** - La GitHub Action actualizará automáticamente `docs.json`

## Formato de Markdown Soportado

- **Negrita** y *cursiva*
- Listas ordenadas y no ordenadas
- Enlaces e imágenes
- Bloques de código
- Tablas
- Citas

## Organización

Los documentos se organizan automáticamente en:
- **Documents**: Vista en árbol por carpetas
- **All Articles**: Todos los artículos ordenados por fecha
- **Recently Read**: Tu historial de lectura

# Getting Started with SimplestColabDocs

SimplestColabDocs is a simple documentation collaboration tool for static sites.

## Main Features

- No external JavaScript libraries
- Tree-like structure with categories
- Full Markdown support
- Light and dark theme
- Full-text search
- Collaboration through Git Pull Requests

## How to Add a Document

1. Add your `.md` file in the `docs` folder (or a subfolder)
2. If your document contains images, place them in a folder with the same name as your document (without .md)
3. **Push to main branch** - The GitHub Action will automatically update `docs.json`

## Supported Markdown Format

- **Bold** and *italic*
- Ordered and unordered lists
- Links and images
- Code blocks
- Tables
- Blockquotes

## Organization

Documents are automatically organized into:
- **Documents**: Tree view by folders
- **All Articles**: All articles sorted by date
- **Recently Read**: Your reading history
